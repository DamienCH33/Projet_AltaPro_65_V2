# AltaPro 65

Site web réalisé pour une entreprise d'élagage et d'entretien d'espaces verts.

L'objectif est de présenter les services proposés par l'entreprise et permettre aux clients de prendre contact.

## Fonctionnalités

* présentation de l'entreprise
* description des services
* formulaire de contact
* interface responsive

## Technologies

* PHP
* HTML
* CSS
* JavaScript
* Bootstrap

## Objectif

Développer un site vitrine professionnel pour une entreprise locale.
